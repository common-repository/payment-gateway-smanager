This plugin adds useful payment links and resources for the payment Forum Plugin to the WordPress Payment Gateway.

== Description ==
It's an online payment system for wordpress users. It is very simple and easy to use. Just install and activate the plugin.

Features:

1. sManager Online Payment is working with almost all banks in bangladesh.
1. No need to use POS machine for your business.
1. You can use this from anywhere.

Activation Process:

* Go to plugin from wp-admin
* Search our plugin with "online-payment-smanager"
* Install the Plugin
* Active the Plugin

Our Support Service is fast and reliable. We always want to see you happy. Customer happiness is our one of the main objectives.

== Installation ==
1. Upload "online-payment-smanager" to the "/wp-content/plugins/" directory.
2. Activate the plugin through the "Plugins" menu in WordPress.
3. Place "do_action( 'plugin_name_hook' );" in your templates.
